Introduction

Python is a cross-platform, object-oriented, high-level programming language. It is a great programming language to learn for beginners as it is simple to use, even though it uses very-high level data storing types, such as arrays and dictionaries. Python is written in a way that is easy to read and more compact than other similar program languages. This is due to the fact that Python includes the following characteristics:

  • The high-level data types allow you to express complex operations in a single statement;
  • Statement grouping is done by indentation instead of beginning and ending brackets;
  • No variable or argument declarations are necessary.

Hello World

To create your first Python code, open your own scrachpad and write your 'Hello World' Python code: print('Hello World')

For this 'Hello World' code, a print() statement is followed by a string. In Python, print() is a function that "prints" the argument that is held within the two parenthesis. The argument, in this case a string, is contained within the () and is what will be shown on the screen after running the code. We will discuss more about strings later on in this document.

Variables

As you can see, there are variables that have data assigned to them utilizing the "=" sign. It's important to note that this does not mean the variable is equal to the data, but simply assigned to it. In Python, it is simple to reassign new data to a variable.
For example, say you already assigned a variable named orange to the integer 6: orange = 6 But you would like to change the number orange is assigned to. You would simply reassign it by setting orange equal to the new integer (or other value): orange = 6
orange = 20 Now orange is assigned to 20.

Numbers

Seen above, you learned that there are two types of number data types in Python:

  • Integers
  • Float

Integers are whole numbers, meaning that they are not superseded by a decimal point. In Python, an integer is denoted by the word int. Floats are numbers comprised of decimal points. Both integers and floats can be utilized with operators to complete math problems.

Strings

In Python, words as datatypes are called strings or str. Strings are enclosed between quotations.

str1 = "This is a string"
Lists

Python knows a number of compound data types, used to group together other values. The most versatile is the list (also called an array), which can be written as a list of comma-separated elements between square brackets. Lists are indexed by a range of numbers that begins with 0. So for example, the first element in the list will have an index of 0. Lists might contain items of different types, but typically the items all have the same type. list_1 = [item1, item2, item3, item4] As mentioned above, a list indexes the elements within it. In the list_1 above, the item2 is at index 1.

Dictionaries

Another useful data type built into Python is the dictionary. Dictionaries are sometimes found in other languages as “associative arrays”. Unlike lists, which are indexed by a range of numbers, dictionaries are indexed by keys, which can be any immutable type; strings and numbers can always be keys. A dictionary syntax is enclosed with curly brackets and its key and value pairs are separated with a colon. Each key,value pair within a dictionary is separated by a comma. dict_1 = {"key1": "orange", "key2": "apple", "key3": "banana"}

Loops

In Python there are two types of loops: for loops and while loops. Loops are utilized to iterate through elements such as lists or dictionaries to find a specific element within that structure.

A for loop iterates over the items of any sequence (a list or a string), in the order that they appear in the sequence. The syntax for a for loop is below: for element in list:
print(element) A for loop will stop running based on the length of the list given. For lists tend to be used for this reason, they typically have a finite end to the loop iteration. It is important to not that the for loop's body needs to be indented one level deeper than the for loop statement.

A while loop, also known as an infinite loop or endless loop, is a loop that runs over and over and does not terminate. In order to prevent a while loop from iterating for eternity, a condition has to be given to the loop. In other words: a while loop runs while the condition runs true. Once the condition is no longer true, the while loop will terminate the iteration. The syntax for a while loop is below: while condition_true: print("Hello, still in the loop!") Here is a code snippet with actual variables to show how while loops require a condition: counter = 0
while counter <= 5:
print(f"Currently at number {counter}.")
counter += 1 In the example above, the counter will be increased by one each time the loop runs. Once the counter integer equals 5, the while loop will terminate because the statement is no longer true.

Conditionals

If statements are used in python to provide a condition to determine what next code block needs to be executed. It can be used to compare variables or to determine what to do with input. The syntax of an if statement is below: if variable == condition:
execute this code
elif variable == other_condition:
execute this code
else:
execute this code

You may notice that there was an elif statement used instead of another if statement. An elif does the same job of an if statement, but will only run if the conditional statement above does not validate as true. You can have as many if and elif statements as you want, however, the first conditional should be an if statement before adding elif statements.
